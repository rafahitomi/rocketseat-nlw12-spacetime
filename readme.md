# NLW 12° Edição - Spacetime

## 🖥️ Projeto

Esse é um projeto Web Responsivo de uma cápsula do tempo, onde o usuário poderá adicionar memórias à uma timeline de acontecimentos marcantes da vida em uma linha do tempo.

## 🚀 Tecnologias

- HTML
- CSS
- Git e GitHub
